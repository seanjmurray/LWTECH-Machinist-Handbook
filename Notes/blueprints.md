# Blueprint reading

### Orthographic Projection

- to be consistent drawing views are arranged 90
- imagine object is inside a glass box then flatten projections onto page "unfolding the box"
- front should be the most informative view/ most details/features
- 6 possible views
- only necessary views should be shown

### Line types

- extension lines: extend from dimension point to dimension
- object line: actual part lines
- leader line: link feature location with detail view on drawing/part
- section line: denotes a cut away section for internal details
- center line: centerline of part/feature
- dimension line: measurements
- phantom line: represent details like threads that are not shown
- hidden line: represent details internal
